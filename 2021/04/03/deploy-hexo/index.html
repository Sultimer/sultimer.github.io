2.搭建过程

//方便上传图片。

图片调整大小：

将_config文件中post_asset_folder: true

_config文件中（注意空格）：

url: https://sultimer.gitee.io/blog：是gitee生成的地址，每次上传以后都需要在git pages中update；

deploy:
type: git

repo: https://gitee.com/sultimer/blog.git
branch: master

gitee中项目名称不需要和用户名一致，但是github中要求二者必须一致，并且项目需要是公开的。

theme: Wikitten

$ cd your-hexo-directory
$ git clone https://github.com/zthxxx/hexo-theme-Wikitten.git themes/Wikitten
然后按照该模板的readme操作

npm install https://github.com/7ym0n/hexo-asset-image –save

npm install hexo-deployer-git –save
